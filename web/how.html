The "How?"

Dear fellow Students,

This is no ordinary fork of the EaglercraftX repl, but a complex proxy to convert TCP to WSS from a powerful backend server. If that makes no sense that's ok, I'm going to go through exactly how this works. This Minecraft server is not actually on Replit but on a hosting service called AeroArk. AeroArk is a vast step from Replit. The CPU is 1.3 times faster, there is 4 times more RAM, and 5 times more storage! This means a almost lagless server. The problem is the server sends data through a protocol called Transmission Control Protocol (TCP) but EaglerCraftX can't read TCP. EaglerCraftX reads a protocol called Websocket (WSS). So I use a BungeeCord Replit server with the EaglercraftXBungee plugin. BungeeCord itself can't do anything besides proxy multiple servers into one. The keyword is "proxy", BungeeCord is a proxy, so the EaglerCraftX plugin takes advantage of it. The plugin uses the BungeeCord proxy to convert the incoming TCP data into WSS and the other way around. Simple yet powerful.
